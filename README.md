#### Contoh praktek dengan app sederhana ####
Install WebSocket
npm init -y
npm install ws
Buat File bernama “server.js”

*** Jalankan server dengan mengetik di CMD
node server.js ***

Sekarang buat file dengan nama “Index.html”

Cara Uji Coba :
Jalankan node server.js
Buka index.html di browser
Buka 2 tab browser
Ketik pesan di salah satu tab
